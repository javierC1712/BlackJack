01

MICROSERVICIOS INDEPENDIENTES

Desarrollado en Java 21 y Spring Boot 3. Cada uno de los 9 módulos escala de forma independiente mediante contenedores Docker aislando fallos y asegurando rendimiento.

02

MENSAJERÍA ASÍNCRONA

Utilizamos Apache Kafka para la comunicación entre servicios. Esto permite liquidaciones de apuestas inmediatas sin generar cuellos de botella en la base de datos PostgreSQL.

03

LICENCIA Y NORMATIVA SCJ

Diseñados desde cero para cumplir la Ley N° 21.562. Incluye módulos estrictos de Juego Responsable y trazabilidad de operaciones para auditorías.

04

PAGOS SEGUROS CON WEBPAY

Integración directa con Transbank. Los retiros y depósitos actualizan el saldo en tiempo real mediante transacciones ACID protegidas.

Factibilidad del Proyecto

NUESTRA PROPUESTA

BlackJack ataca directamente el problema de latencia y cumplimiento legal que sufren los operadores extranjeros en Chile.

🛡️
FACTIBILIDAD LEGAL

El proyecto cumple con la Ley N° 21.562 integrando módulos de verificación (KYC) y sistemas de juego responsable regulados.

⚡
FACTIBILIDAD TÉCNICA

Arquitectura basada en Spring Boot, Docker y Kafka, escalable para soportar grandes picos de usuarios concurrentes.

🎲
FACTIBILIDAD COMERCIAL

Ofrecemos baja latencia en mercados locales, resolviendo el principal problema competitivo frente a plataformas extranjeras.
